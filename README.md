# Project4_NLP

A js project built using webpack, express, and the meaningcloud API that allows for user to search a url and evaluates the blogs written within based on subjectivity, confidence and irony.

# Dev Build Script

```python
npm run build-dev
```

# Express Server

```python
npm run start
```

# Jest

```python
npm run test
```

# Production Build

```python
npm run build-prod
```
